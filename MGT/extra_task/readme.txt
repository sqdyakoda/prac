Инструкция
Чтобы продемонстрировать решение, нужно:

1. Перейти по ссылке https://labs.play-with-docker.com/ и войти в свою учетную запись docker
2. Нажать на кнопку старт (начать сессию)
3. нажать на кнопку "+ADD NEW INSTANCE" слева. 
4. В открывшемся терминале нужно создать файл docker-compose.yml с помощью команды:
    touch docker-compose.yml
5. Перейти в EDITOR (кнопка находится над терминалом)
6. Нажать на файл docker-compose.yml. Откроется текстовый редактор
7. Вставить туда следующий код:

services:
  server:
    image: yakoda/root-server:latest
    ports:
      - "8080:8080"
  client:
    image: yakoda/root-client:latest
    depends_on:
      - server

8. Сохранить файл, нажав на кнопку "Save"
9. Перейти обратно на вкладку с терминалом и запустить программу в терминале с помощью команды:
    docker-compose up

Примечание:
В директории ./client/data/ находятся txt-файлы с входными данными и файл list.txt, в каждой строке которого и указаны имена этих файлов.
Клиент берёт входные данные только из файлов, имена которых находятся в list.txt
